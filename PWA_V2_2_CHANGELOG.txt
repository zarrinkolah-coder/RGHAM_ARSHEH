نسخه ۲٫۲ — اتصال PWA نصب‌شده به سرور مستقل

- افزودن صفحه «اتصال سرور» فقط برای مدیر
- امکان ثبت آدرس HTTPS سرور و کلید API روی خود گوشی
- استفاده از API سرور مستقل به جای مسیر ثابت همان دامنه
- دریافت خودکار اطلاعات جدید سرور هر ۸ ثانیه
- ارسال تغییرات گوشی به SQLite سرور
- پشتیبانی CORS کنترل‌شده برای GitHub Pages
- افزودن RAGHAM_API_TOKEN برای محافظت API
- حفظ حالت محلی در صورت غیرفعال بودن سرور
- جلوگیری از پیام‌های تکراری در حالت محلی
- افزایش نسخه Service Worker به ۲٫۲ برای دریافت بروزرسانی
