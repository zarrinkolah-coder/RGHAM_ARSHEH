نسخه 1.1.0
- افزودن ذخیره فایل پشتیبان JSON فقط در نمای مدیر
- افزودن انتخاب فایل و بازیابی JSON روی پایگاه داده مرکزی سرور
- نمایش خلاصه محتوای فایل پیش از تأیید بازیابی
- دانلود خودکار پشتیبان وضعیت فعلی پیش از بازیابی
- نگهداری خودکار نسخه قبلی در جدول state_backups در SQLite
- پشتیبانی از فایل‌های پشتیبان جدید و فایل‌های JSON قدیمی برنامه

نسخه سروری مرحله اول نرم‌افزار رقم

- اجرای برنامه روی Linux، Windows Server و Docker
- ذخیره مرکزی داده‌ها در SQLite
- API همگام‌سازی داده با Patch ردیفی
- کنترل تعارض نسخه هنگام ذخیره همزمان
- بروزرسانی خودکار مرورگرها هر 8 ثانیه
- نگهداری نسخه‌های پشتیبان داخل پایگاه داده
- ابزار پشتیبان‌گیری مستقل SQLite
- ابزار ورود فایل JSON موجود به پایگاه داده
- پشتیبانی از Basic Auth کلی اختیاری
- نمونه تنظیم Nginx و systemd
