# گزارش بازنگری نسخه PWA نرم افزار رقم

نسخه ۲٫۰ بر پایه نسخه سروری و رابط نهایی برنامه تهیه شده است. پروژه Android WebView قبلی برای PWABuilder مناسب نبود، زیرا PWABuilder یک نشانی وب PWA را بسته‌بندی می‌کند، نه پروژه Gradle را.

## اصلاحات انجام‌شده

- افزودن `manifest.webmanifest` کامل با نام، توضیح، زبان فارسی، راست‌به‌چپ، رنگ‌ها، آیکون‌های عادی و maskable، تصاویر و میانبرها.
- افزودن Service Worker با کش پوسته برنامه، حالت آفلاین و بروزرسانی کنترل‌شده.
- افزودن دکمه نصب PWA و اعلان آماده‌شدن نسخه جدید.
- افزودن نمایش وضعیت قطع ارتباط و نگهداری تغییرات انجام‌شده روی دستگاه تا اتصال مجدد.
- جلوگیری از کش‌شدن درخواست‌های API و فایل‌های حساس.
- حذف وابستگی فونت آنلاین برای اجرای پایدارتر و آفلاین.
- جداکردن CSS و JavaScript از HTML برای نگهداری، کش و بررسی نحوی بهتر.
- افزودن آیکون‌های ۱۹۲ و ۵۱۲ پیکسل، آیکون‌های maskable و Apple Touch Icon.
- افزودن تصاویر واقعی تلفن همراه و دسکتاپ برای Manifest.
- اصلاح محدودسازی تعداد نسخه‌های داخلی هنگام بازیابی کامل اطلاعات در SQLite.
- افزودن MIME صحیح برای Web Manifest و JavaScript.
- افزودن سرآیندهای امنیتی، سیاست مجوزها و سیاست کش مناسب.
- افزودن مسیر نسخه و سلامت سرور.
- افزودن تست Smoke و GitHub Actions برای بررسی Python، JavaScript، Manifest، Service Worker و API.
- افزودن `.gitignore` تا پایگاه داده، بکاپ‌ها، رمزها و فایل‌های موقت وارد GitHub نشوند.
- افزودن پوشه `.well-known` و راهنمای `assetlinks.json` برای Trusted Web Activity اندروید.

## محدودیت امنیتی باقی‌مانده

ورود کاربران و سطح دسترسی رکوردها هنوز عمدتاً در رابط مرورگر کنترل می‌شود. برای قرار دادن اطلاعات واقعی در اینترنت عمومی، احراز هویت و مجوزدهی کامل سمت سرور باید در مرحله بعد توسعه یابد. تا آن زمان استفاده داخلی، VPN یا لایه محافظ معکوس توصیه می‌شود.
